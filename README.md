# Dancing Digital network graphs

Create network graphs for Dancing Digital. [Live demo](https://wykhuh.github.io/wikibase-search/).

This app is built using SvelteKit.

## Setup

1. install librariers

```
npm install
```

2. run the server

```
npm run dev
```

## Code checking

run tests

```
npm run tests
```

format & lint

```
npm run format
npm run lint
```
